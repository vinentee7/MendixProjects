# MendixProjects
Projetos criados com Mendix em parceria com a empresa Mirai, GO Explosion e Residência Porto Digital (UCB)

Durante os desafios apresentados, o foco principal foi demonstrar como o Mendix pode acelerar o desenvolvimento e facilitar a criação de soluções empresariais. No Desafio Mirai, a proposta foi construir um site para gerenciamento de contas, destacando como a plataforma permitiu agilizar a produção, uma funcionalidade que queriamos adicionar é a verificação de splits, como uma página de validação antes da aprovação final.

No Desafio Porto, o objetivo foi criar uma página de cadastro para novos assinantes de newsletter. Novamente, o Mendix se mostrou vantajoso, acelerando o desenvolvimento, outra funcionalidade desejada era um perfil de administrador para envio de atualizações.

Em ambos os casos, a ferramenta se destacou por reduzir o tempo de entrega e simplificar o processo de implementação de funcionalidades.
